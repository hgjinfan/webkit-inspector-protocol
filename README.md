- [源1](https://github.com/WebKit/WebKit/tree/main/files/Source/JavaScriptCore/inspector/protocol)
同步位置：/protocol
- [源2](https://github.com/WebKit/WebKit/tree/main/Source/WebInspectorUI/Versions)
同步位置：/versions
- 同步方式：GitHub Actions，每天 00:00 UTC（见 `.github/workflows/sync-webkit-protocol.yml`）
